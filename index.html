In contrast to video-based rollout and point-based scene-flow prediction, which may suffer from visual artifacts, shape drift, or object-level inconsistency, ORD-WM performs multi-step prediction by analytically applying predicted object-wise SE(3) updates to geometry-aware object latents composed of 3D vector features, preserving object geometry while updating pose. This latent-space rollout enables real-world object motion prediction and closed-loop model predictive control (MPC), where candidate actions are evaluated by their predicted effects on object states.

Abstract

World models provide robots with a predictive mechanism for reasoning about how actions change object states in manipulation tasks. However, existing image-based and dense 3D world models do not explicitly impose object-level rigid motion, leaving rigid-object geometry preservation to be learned implicitly and making multi-step rollouts susceptible to shape drift or object-level motion inconsistency. To address this, we propose ORD-WM, a 3D world model that learns object-centric rigid dynamics for robotic manipulation. ORD-WM learns a vector-neuron variational autoencoder supervised by signed distance fields, which encodes each object point cloud as a geometry-aware latent representation composed of 3D vector features, and uses an action-conditioned dynamics model to predict object-wise SE(3) motion increments. Rather than generating unconstrained next latent states, ORD-WM analytically applies predicted object-wise rigid transformations to the current latent states, separating persistent object geometry from pose changes. Experiments show that ORD-WM enables stable, geometry-preserving multi-step prediction and supports efficient real-time replanning for closed-loop model predictive control in simulated and real-world manipulation scenarios. More details and videos are available at: https://anonymous.4open.science/w/ORD-WM/.
